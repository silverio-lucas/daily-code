# daily-code

Repositório de código diário focado em resolução de problemas pequenos, testes e prática deliberada.

## Objetivo

- Registrar prática diária de código  
- Construir repertório de soluções reutilizáveis  
- Evoluir de snippets simples para padrões sólidos  

## Estrutura

daily-code/
  01.html
  02.js
  ...

(Estrutura inicial simples — será evoluída)

## Como usar este repositório

Cada arquivo representa:

- Um problema específico resolvido  
- Uma solução direta (sem overengineering)  
- Um passo incremental na prática diária  

## Próximos passos

- Organizar por categorias:
  - html/
  - css/
  - javascript/
  - node/

- Evoluir para:
  - patterns/
  - utils/
  - experiments/

- Adicionar README nos snippets relevantes

## Filosofia

Volume + consistência > perfeição.

Este repositório registra processo, não apenas resultado.

## Futuro

Os melhores códigos serão extraídos para repositórios mais estruturados (patterns, libs, projetos maiores).
